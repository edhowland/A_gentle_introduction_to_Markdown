Links

Links are span elements and can occur within paragraphs or within list elements, .etc.

Normal links

A link consists of a text description followed by the URL. The description is enclosed within square brackets: '[ ... ]'. The URL is enclosed within parenthesis : '( ... )'

A title or hover tip can be added to a link by enclosing it in double quotes after the URL but before the closing right paren

Examples

- [Example Web site:http://www.example.com](http://www.example.com)

- [This presentation:: https://github.com/edhowland/A_gentle_introduction_to_Markdown](https://github.com/edhowland/A_gentle_introduction_to_Markdown)
- [The Arch Wiki](https://wiki.archlinux.org "A great place for Linux information")

Becomes:

  • Example Web site:http://www.example.com
  • This presentation:: https://github.com/edhowland/AgentleintroductiontoMarkdown
  • The Arch Wiki

Bare links

If you want links which have the text is the same as the URL, you can use this style:

<http://www.example.com>

It will appear as:

http://www.example.com
