Lists

Lists are block elements and can be ordered or unordered lists. Lists can also be nested.

Adding some list elements

Unordered lists

Unordered lists can be added with sequential lines beginning with either '-', '*' or '+' Nested lists can use a combination of these types.

- List element
- Another list element
- Yet another list element
- Final list element

Becomes:

  • List element
  • Another list element
  • Yet another list element
  • Final list element

Ordered lists

Ordered lists use numbers followed by a period: '.' They can be out of sequence, but must start with : '1.'

1. List number 1
3. List number 2
4. List number 3
2. List number 4

Becomes:

  1. List number 1
  2. List number 2
  3. List number 3
  4. List number 4
